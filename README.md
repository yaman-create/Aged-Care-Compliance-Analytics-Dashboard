# 🇦🇺 Australian National Aged Care Compliance & Risk Dashboard

## 📌 Project Overview
An end-to-end data analytics and business intelligence project built using official Australian Government GEN Aged Care data. The project bridges the gap between raw public administration records and actionable clinical insights by simulating high-risk operational metrics.

## 📊 Key Features & Analytics
* **Granular LGA-Level Insights:** Analyzed residential care distribution across 500+ Local Government Areas.
* **Clinical Risk Profiling (Falls Rate):** Designed a predictive risk metric to identify facilities with high quarterly fall counts using whole-number rounding optimization.
* **Regulatory Compliance Tracking:** Built dynamic monitoring for the government's 200-minute average daily care benchmark.
* **Patient Workload Correlation:** Side-by-side comparison of Dementia loads against Polypharmacy (multi-medication) risks.

## 🛠️ Tech Stack & Excel Functions Used
* **Data Enrichment:** `RANDBETWEEN`, `ROUND`, `Nested Multiplication`
* **Data Aggregation:** Dynamic Pivot Tables with custom Value Field Settings (Averages & Sums)
* **UI/UX & Business Intelligence:** Clustered Bar Charts, Line Charts with adjusted scale parameters, Interactive Slicers with multi-pivot Report Connections, Custom Flat-UI Color Palettes.

## 🚀 How to Interact
1. Download the `.xlsx` file from this repository.
2. Open in Microsoft Excel (Mac/Windows).
3. Navigate to the `Dashboard` tab and use the **LGA Slicer** on the right to filter clinical insights live.
